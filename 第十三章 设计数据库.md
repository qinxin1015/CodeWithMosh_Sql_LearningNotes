# 第十三章 设计数据库

### 13.2 数据建模
数据建模，就是为要存储在数据库中的数据 创建模型的过程
数据建模包括四个步骤
- 1. 理解和分析业务需求。  最重要，但经常被忽视的一步
- 2. 构建关于业务的**概念模型**。 包括识别业务中的实体、事务或概念以及他们之间的关系
- 3. 构建**逻辑模型**。 生成一个数据模型或数据结构用以存储数据
- 4. 为特定的数据库管理系统构建一个**实体模型**。 实体模型是围绕数据库技术的逻辑模型的实现
